# MealPlan Home Assistant Add-on

Mobile-first meal planning application for Home Assistant.

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]
![Supports armv7 Architecture][armv7-shield]

## About

MealPlan helps you plan your weekly meals with an intuitive drag-and-drop interface, optimized for mobile devices.

## Installation

1. Add the repository to Home Assistant:
   - Navigate to Settings → Add-ons → Add-on Store
   - Click the three-dot menu → Repositories
   - Add: `https://github.com/jhaals/mealplan`

2. Install the MealPlan add-on

3. Start the add-on

4. Access at `http://homeassistant.local:3001`

## Configuration

```yaml
port: 3001
```
